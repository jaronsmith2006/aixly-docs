# Docs site (Docusaurus)

This folder contains the **Docusaurus site** for merchant help docs.

## Where the docs content lives

The actual markdown articles live in the repo root:

- `../docs`

This Docusaurus site is configured to read docs from that directory.

## Run locally

From `docs-site/`:

```bash
npm install
npm run start
```

Then open the URL printed in your terminal (usually `http://localhost:3000`).

## Build

```bash
npm run build
npm run serve
```
