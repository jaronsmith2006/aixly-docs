## 1) Install the app

- Install the app in Shopify and approve permissions.
- After install, open it from **Shopify Admin → Apps**.

---

## 2) Follow the in-app Setup Guide

In the app, go to:

- **Dashboard → Overview**

The **Setup Guide** walks you through:

- **AI processing**: initial training progress
- **Enable storefront widget**: opens your theme editor
- **Verify launcher on storefront**: confirms the widget is live
- **Choose a plan**

---

## 3) Enable the storefront chat widget (Theme App Embed)

In Shopify admin:

- Go to **Online Store → Themes**
- Click **Customize**
- Open **App embeds**
- Enable **Aixly AI Chatbot**
- Click **Save**

---

## 4) Verify it’s live

You should see the chat launcher on your storefront.

If you don’t:

- Go back to the app → **Dashboard → Overview**
- In the Setup Guide, click **Refresh status**

If it still doesn’t appear, see: **Troubleshooting → Common issues**.
