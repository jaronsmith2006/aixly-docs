## Where to find it

- In the app, open **Appearance**

Use the right-side preview to see changes instantly, then click **Save settings**.

---

## Themes

Use **Themes** to start from a preset and adjust from there.

---

## Colors

Adjust:

- Accent color (launcher + header)
- Assistant bubble color
- Assistant text color

---

## Window (header)

Configure:

- Title
- Subtitle
- Avatar image URL (optional)

---

## Button (launcher)

Configure:

- Position (left/right)
- Shape (circle/square)
- Button size
- Icon (preset icons or upload your own)
- Button label and corner radius
- Show/hide launcher

---

## Layout

Configure:

- Widget size (sm/md/lg)
- Widget corner radius
- First message
- Input placeholder
